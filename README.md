# Person-Tracking-Robot

- Goal of project is to produce robot that will follow a person at a set distance, while person is walking around.
- Project utilizes Yolov8 as the tracking program
- to run project start Main file with other files in director, main file is a multithreading file while facilitates everything and the order to run
- Led indicator meaning
- Red - no traget
- Green - sees target
- blue - hand gesture seen
